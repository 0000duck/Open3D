Reconstruction system
===================================================================

.. _reconstruction_system_overview:

This tutorial introduces a pipeline reconstructs a 3D scene from RGBD sequence. The basic idea of the pipeline is inspired from [Choi2015]_, and several ideas introduced in [Park2017]_ are adopted in this pipeline for better reconstruction results.
